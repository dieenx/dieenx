>tải spotify mod
```powershell
iex "& { $(iwr -useb 'https://raw.githubusercontent.com/SpotX-Official/spotx-official.github.io/main/run.ps1') } -confirm_uninstall_ms_spoti -confirm_spoti_recomended_over -podcasts_off -block_update_on -start_spoti -new_theme -adsections_off -lyrics_stat spotify"
```
>tải discord và vencord
```powershell
$folder = "$env:USERPROFILE\Downloads"
curl.exe -L --progress-bar "https://discord.com/api/downloads/distributions/app/installers/latest?channel=stable&platform=win&arch=x64" -o "$folder\DiscordInstaller.exe"
curl.exe -L --progress-bar "https://github.com/Vencord/Installer/releases/latest/download/VencordInstaller.exe" -o "$folder\VencordInstaller.exe"

```
